# Description

This is a project that consists on a quiz made using the following topics:

<ul>
  <li> <a href="https://reactjs.org/"> ReactJs </a> </li>
  <li> <a href="https://nextjs.org/"> NextJs </a> </li>
  <li> <a href="https://styled-components.com/"> Styled-components </a> </li>
  <li> <a href="https://www.npmjs.com/package/prop-types"> Prop-types </a> </li>
  <li> <a href="https://www.framer.com/motion/"> Framer Motion  </a> </li>
  <li> <a href="https://react-icons.github.io/react-icons/"> React Icons </a> </li>
</ul>

The [interface prototype](https://www.figma.com/file/cg1MIzSRRss8ggpypQbmdD/AluraQuiz?node-id=0%3A1) was made in figma by the [Alura team](https://www.alura.com.br/) for the course "Imers√£o React Next".
